EZ Spark Parent
===
### EZ Spark 构建框架，可使用Scala编写Spark程序基础包。

##使用

    <dependency>
        <groupId>com.ecfront</groupId>
        <artifactId>ez-spark-parent</artifactId>
        <version>0.1.0</version>
    </dependency>

### Check out sources
`git clone https://github.com/gudaoxuri/ez-spark-parent.git`
